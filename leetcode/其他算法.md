## 厄拉多塞筛法

可以用来统计 [2,n] 中质数的数量。

如果 $x$是质数，那么$x$的倍数一定不是质数，从2开始，从小到大遍历每个数，将其所有的倍数都标记为合数（除了2自己），即0，然后从3开始标记........

#### 代码

```c++
class Solution {
public:
    int countPrimes(int n) {
        vector<int> isPrime(n, 1);
        int ans = 0;
        for (int i = 2; i < n; ++i) {
            if (isPrime[i]) {
                ans += 1;
                if ((long long)i * i < n) {
                    for (int j = i * i; j < n; j += i) {
                        isPrime[j] = 0;
                    }
                }
            }
        }
        return ans;
    }
};
```
